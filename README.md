# SK Laundry Services

An enterprise-level Full Stack Laundry Management Platform featuring a Customer Portal, Online Ordering, Live Booking Tracking, UPI Payments Integration, and an Owner-Only Admin Dashboard.

## 🚀 Features

### 💻 Customer Experience
- **Online Booking / Order Now**: Seamlessly place laundry/dry-cleaning orders with garment details and custom requests.
- **Garment Images Upload**: Upload images of your clothes during checkout for verification.
- **Track Order / Live Progress**: Real-time status updates ranging from Pickup, Washing, Drying, Quality Check, to Out for Delivery and Delivered.
- **UPI QR Code Payments**: Safe, secure, and modern UPI integration with instant transaction logging.
- **Doorstep Addresses**: Save and reuse addresses for easy pickups and deliveries.

### 🛡️ Owner & Admin Suite
- **Secure PIN Authentication**: Safe owner login with session integrity.
- **Two-Step Password Change**: Security & PIN portal requiring current password verification before setting a new one.
- **Pricing Editor**: Update rates dynamically for different garment types and service tiers.
- **Booking Ledger**: Manage all incoming, ongoing, and completed laundry batches.
- **Customers Roster & Sales Reports**: Audit customer data, search records, and view high-level revenue charts.
- **Audit Logs**: Transparent security tracing for administrative operations.

## 🛠️ Tech Stack
- **Frontend**: React 18+, TypeScript, Tailwind CSS, Lucide Icons, Framer Motion
- **Backend**: Express (Node.js/TypeScript)
- **Tooling**: Vite, esbuild
